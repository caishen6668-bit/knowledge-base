## 🎯 使用建议

!!! tip "让知识库发挥最大价值"
    1. **随手记录** — 遇到有价值的信息立即记下
    2. **定期整理** — 每周花 10 分钟分类整理
    3. **关联思考** — 用链接将不同知识点串联
    4. **复习回顾** — 定期翻阅旧笔记加深记忆
