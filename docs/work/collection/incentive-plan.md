# 催收团队激励方案

## 一、方案背景

为了提升催收团队整体回款率与阶段达成率，同时强化组别间竞争与协作，特制定本激励方案。奖励以**组别为单位**，组长统一管理奖金使用（团建、采购奖励物品等）。

- **适用团队**：新客组（A/B/C）、老客组（D/E）
- **奖励周期**：半月度或周度
- **考核目标**：各阶段回款达成率及加权达成率

## 二、考核原则

| 原则 | 说明 |
|:---|:---|
| 组别为单位 | 组长负责分配团队奖金 |
| 阶段目标明确 | D0、D1、S1 三个阶段各有独立目标 |
| 加权平均分计算 | 根据阶段权重计算组别加权平均分（100 分制），超过 **100 分**才有机会获得奖励 |
| 排名制奖励 | 新客组 ABC 只奖励排名第一；老客组 DE 只奖励排名第一；其他组别无奖励 |

## 四、加权平均分计算方法

```
阶段达成率 = 回款阶段金额回收率 / 回款阶段目标
阶段得分   = 阶段达成率 × 阶段权重
组别加权平均分 = D0得分 + D1得分 + S1得分
```

**达标判定**：加权平均分 > 100 分 → 可参与排名；≤ 100 分 → 不得奖

## 五、奖励发放规则

- 各新客组加权平均分排名，第一名获奖励
- 各老客组加权平均分排名，第一名获奖励
- 奖金由组长自主分配给团队，可用于团建或奖励物品
- 未排名组：无奖励
