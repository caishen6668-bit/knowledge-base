# 外包机构催记规范 / Normativas para la tipificacion

## 1. 记录间隔 / Intervalo entre tipificaciones

每个案件的催记记录间隔不得超过 **2 小时**，无论何种联系方式。

## 2. 记录方式 / Metodo de tipificacion

所有催记均须通过系统预设的**下拉菜单**录入，确保选项正确，数据标准化。

**选择顺序**：
```
与客户关系 → 联系方式 → 联系状态 → 联系结果
```

## 3. 关键记录项 / Punto clave de tipificacion

### 联系方式 / Metodo de contacto
请选择实际使用的联系手段（电话、短信、WhatsApp、邮件等）。

### 联系状态 / Estado del contacto
选择如"接通"、"未接"、"空号/停机/关机"、"语音信箱留言"等，要求准确反映实际情况。

### 联系结果 / Resultado del contacto
选择标准结果，如"有还款意愿"、"继续跟进"、"没钱还款"、"客户欺诈"等。若无合适选项，请选择"其他"并在备注中说明。

## 4. 补充要求 / Requisitos adicionales

- 系统将自动监控记录间隔和选项使用情况
- 选项错误或记录遗漏将直接影响**数据统计与绩效考核**
- 如遇特殊情况无法满足标准要求，必须在备注中详细说明原因
