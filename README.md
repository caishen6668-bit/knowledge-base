## 🚀 快速开始

### 环境准备

```bash
# 1. 克隆仓库
git clone https://github.com/caishen6668-bit/knowledge-base.git
cd knowledge-base

# 2. 创建虚拟环境
python -m venv .venv

# 3. 安装全局依赖
.venv/Scripts/pip install -r requirements.txt

# 4. 各项目依赖按需安装（见各项目 README.md）
```

### 启动本地文档

```bash
.venv/Scripts/mkdocs serve
# 浏览器打开 http://127.0.0.1:8000
```

## 📝 规范

### 新增项目

1. 在 `项目/` 对应业务目录下创建项目文件夹（中文名）
2. 每个项目必须包含 `README.md`（模板：[项目简介、项目结构、运行方式、输入、输出、依赖、修改记录、注意事项、负责人]）
3. 每个项目维护自己的 `requirements.txt`
4. 真实业务数据不上传 GitHub（已在 `.gitignore` 排除）

### 归档项目

项目不再维护时，移至 `已归档项目/`，并添加归档说明 README。

### 提交规范

```bash
git add .
git commit -m "描述本次变更"
git push
```

## 🔒 安全说明
