# 业务口径 — 金额首逾率

> 已于 UAT 与 Quick BI 验证一致（2026-07-06）。

---

## D0 金额首逾率

### 到期本金

```
due_amt
```

`due_amt` = `D_2_due_amt` + `D_3_pay_amt`

> 注：`due_amt` 是订单最初应还总额，不随阶段变化。同一订单在所有阶段（D-2 / D-1 / D0 / D1 / S1 / S2）的到期本金相同。

### 累计回款

按阶段递增累计：

| 阶段 | 累计回款公式 |
|------|-------------|
| D-2 | `D_3_pay_amt + D_2_pay_amt` |
| D-1 | `D_3_pay_amt + D_2_pay_amt + D_1_pay_amt` |
| **D0** | `D_3_pay_amt + D_2_pay_amt + D_1_pay_amt + D0_pay_amt` |
| D1 | `... + D1_pay_amt` |
| S1 | `... + S1_pay_amt` |
| S2 | `... + S2_pay_amt` |

### 金额首逾率

```
金额首逾率 = 1 - 累计回款 / due_amt
```

### D0 公式

```
金额首逾率(D0) = 1 - (D_3_pay_amt + D_2_pay_amt + D_1_pay_amt + D0_pay_amt) / due_amt
```
