# ⚡ 常用命令速查

## Git

```bash
# 克隆仓库
git clone <repository-url>

# 查看状态
git status

# 添加文件
git add .
git add <file>

# 提交
git commit -m "提交信息"

# 推送到远程
git push origin main

# 拉取更新
git pull origin main
```

## MkDocs

```bash
# 启动本地服务
mkdocs serve

# 构建静态站点
mkdocs build

# 部署到 GitHub Pages
mkdocs gh-deploy
```

## Python

```bash
# 安装依赖
pip install <package>

# 导出依赖
pip freeze > requirements.txt

# 安装依赖（从文件）
pip install -r requirements.txt
```
