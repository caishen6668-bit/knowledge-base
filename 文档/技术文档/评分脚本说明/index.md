# 💻 绩效评分系统 - Python 脚本

催收团队绩效评分的自动化工具集，用于计算员工业绩、出勤、入职、合规分数，生成周度/月度排名和评级。

## 系统架构

```
原始数据 (Excel)
    │
    ▼
┌─────────────────┐
│  各维度计算模块  │
├─────────────────┤
│ 业绩分.py        │  ← 计算回款达成率得分
│ 出勤分.py         │  ← 计算缺勤扣分
│ 入职分.py         │  ← 计算入职月数加分
│ 业绩+出勤.py      │  ← 合并业绩和出勤
└────────┬────────┘
         ▼
┌─────────────────┐
│  汇总排名模块    │
├─────────────────┤
│ 员工最终评分.py   │  ← 综合四维度得出最终评分
│ 组员等级排名.py   │  ← 生成等级排名
│ 周度人员排名.py   │  ← 周度排名表
│ 周度日度生成.py   │  ← 周度/日度报表
└────────┬────────┘
         ▼
┌─────────────────┐
│  激励计算模块    │
├─────────────────┤
│ 个人激励原.py     │  ← 个人激励原始版本
│ 个人激励提200.py  │  ← 个人激励提成（200版本）
│ xby每日.py        │  ← 每日数据处理
└─────────────────┘
```

## 使用流程

1. 准备原始 Excel 数据文件
2. 按顺序运行各维度计算脚本
3. 运行汇总脚本生成最终评级
4. 输出结果到 Excel
