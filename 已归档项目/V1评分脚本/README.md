# V1 评分脚本（已归档）

## 归档说明

这些脚本是员工评级系统的 **V1 版本**，已被 [员工评级系统 V2.1](../../项目/催收业务/墨西哥/员工评级系统/README.md) 完全替代。

### 归档原因

| V1 | V2 |
|-----|-----|
| 数据从本地 Excel 手动导入 | 直接从 Quick BI API 自动取数 |
| 每个评分维度独立脚本运行 | 一条命令完成全部流程 |
| 无缓存，每次需重新准备数据 | 自动缓存，秒级重复运行 |
| 输出仅中文 Excel | 输出中文 + 墨西哥西语双版本 |

### 替代方案

请使用 `../../项目/催收业务/墨西哥/员工评级系统/`（V2.1.1）。

```bash
cd D:/knowledge-base
python -m 项目.催收业务.墨西哥.员工评级系统.main
```

## 归档内容

| 脚本 | 原用途 |
|------|--------|
| `业绩分.py` | 从 Excel 计算员工业绩达成率 |
| `出勤分.py` | 从出勤 Excel 计算出勤得分 |
| `入职分.py` | 从花名册计算工龄得分 |
| `业绩+出勤.py` | 合并业绩和出勤数据 |
| `员工最终评分.py` | 加权计算最终综合评分 |
| `组员等级排名.py` | 按评分排名分等级 |
| `6月冲刺奖励.py` | 6 月冲刺奖励活动分析 |
| `预警名单.py` | 标注需预警的员工名单 |
